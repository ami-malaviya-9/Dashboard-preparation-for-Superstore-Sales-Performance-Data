# Dashboard-preparation-for-Superstore-Sales-Performance-Data
An interactive Super Store Sales Dashboard built using Power BI, designed to analyze sales performance, profitability, and customer purchasing patterns across different regions, categories, and time periods.

## Project Overview

This project presents an interactive Super Store Sales Dashboard built using Power BI, designed to analyze sales performance, profitability, and customer purchasing patterns across different regions, categories, and time periods.

The dashboard enables stakeholders to monitor business performance, identify growth opportunities, and make data-driven decisions.

This project demonstrates skills in data visualization, business intelligence, and sales analytics using Power BI.

## Objectives

Analyze overall sales and profit performance

Track regional and segment-wise trends

Identify top-performing products and categories

Evaluate customer purchasing behavior

Monitor key performance indicators (KPIs)

Support strategic business decisions

## Dashboard Features
### Sales Overview

Total Sales

Total Profit

Total Orders

Profit Margin

Year-over-Year performance

### Regional Analysis

Sales by Region

Profit by State or City

Geographic performance comparison

Regional contribution to total revenue

### Product Performance

Sales by Category and Sub-Category

Top-selling products

Most profitable products

Impact of discounts on profitability

### Customer Segment Analysis

Sales by customer segment (Consumer, Corporate, Home Office)

Segment-wise profitability

Purchase behavior patterns

### Time-Based Trends

Monthly and yearly sales trends

Seasonal patterns

Growth analysis over time
